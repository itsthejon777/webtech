Windows 10 en 11 implementatie

WDS (Windows Deployment Services) was een van mijn eerste grote projecten en draaide om de implementatie van Windows Server 2022 voor het massaal deployen van Windows-besturingssystemen. Het doel was om een efficiënt systeem te creëren waarmee ik meerdere computers tegelijkertijd kon voorzien van een gestandaardiseerde versie van Windows. Voor dit project maakte ik gebruik van port forwarding om via een lokaal netwerk toegang te krijgen tot de machines van andere studenten, zodat ik snel en eenvoudig de benodigde software kon installeren en beheren.

Het project begon met het opzetten van een Windows Server 2022-omgeving, waar ik verschillende essentiële netwerkservices configureerde, zoals DNS, DHCP, en WDS zelf. Deze services waren cruciaal om een netwerkgebaseerde installatie mogelijk te maken. Vervolgens creëerde ik geïmageerde versies van zowel Windows 10 als Windows 11, compleet met vooraf geïnstalleerde applicaties die nodig waren voor de dagelijkse werkzaamheden van de studenten. Door middel van deze images kon ik de computers snel en consistent voorzien van de juiste software. Het deployen zelf gebeurde via Ethernet, waarbij ik de geïmageerde versies naar de doelmachines stuurde. Dit proces bespaarde niet alleen tijd, maar zorgde ook voor een uniforme installatie op alle systemen, wat essentieel was voor de efficiëntie en het beheer van het netwerk. Daarnaast heb ik een uitgebreide stap-voor-stap handleiding gemaakt waarin ik uitleg hoe het hele proces werkt. Deze handleiding kan worden geraadpleegd via de volgende link: Google Docs handleiding. Het project gaf me waardevolle ervaring in het werken met Windows Server en netwerkinfrastructuren, en het was een belangrijke stap in mijn ontwikkeling als IT-professional.
